Backend Notes for James

GET /api/seo/gsc → Google Search Console API v1 (searchanalytics.query). Requires OAuth scope: https://www.googleapis.com/auth/webmasters.readonly. Pull clicks, impressions, CTR, position per keyword. Store in Supabase seo_keywords table (company_id, keyword, clicks, impressions, position, date). GET /api/seo/gbp → Google Business Profile API (businessInformation.locations.get). Pull views, searches, direction requests, phone calls. Store in Supabase gbp_metrics table. Both sync daily via Railway cron.
